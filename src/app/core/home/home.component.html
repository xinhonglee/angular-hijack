THE GAME
SuperOne offers unique, exciting and rewarding experiences while buying and using playballs to hunt for huge hidden prizes.
Game mechanics are built around innovative bidding algorithms. Combined with player skill, a fresh new crypto prize game is created.

THE TOKENS
Game profits are shared via tokens. Tokens are sold in blockchain spots for 0.1 ETH each and contain a decreasing amount of tokens.
Tokens are minted in real time via smart contract, providing instant, direct ownership and access.
